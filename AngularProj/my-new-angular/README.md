# Angular Project
## Pre-requisites
### node verions
```sh
node -v
```
### npm verions
```sh
npm -v
```
### create project
```sh

npm install -g @angular/cli@21
```
### Angular version
```sh
ng version
```
## Initialize project
### create app
```sh
ng new my-angular-app
```
### Serve app
```sh
cd my-angular-app
ng serve
```

### Structure
- app.html

- app.css
- app.ts

### 3 brackets
- {{}} data from ts -> html
- () event from html -> ts
- [] html properties tag
## Add component
### new component
```sh
ng generate component <component-name>
```
